(slider! entity k & options)

Calls a single method on a slider.

Options
:get-style
Returns the slider's style. Modifying the returned style may not have an effect until {@link #setStyle(SliderStyle)} is called.
:is-dragging
Returns true if the slider is being dragged.
:set-style^Slider.SliderStyle style
:set-visual-interpolation-inverse^Interpolation interpolation
Sets the inverse interpolation to use for display. This should perform the inverse of the {@link #setVisualInterpolation(Interpolation) visual interpolation}.
Source
(defmacro slider!
  [entity k & options]
  `(u/call! ^Slider (u/get-obj ~entity :object) ~k ~@options))
