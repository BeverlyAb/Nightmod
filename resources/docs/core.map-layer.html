(map-layer screen layer & options)

Returns a MapLayer from the tiled map in screen that matches layer. This is necessary for non-tile layers, like object and image layers.

(map-layer screen "objects")
Options
:get-name
:get-objects
:get-opacity
:get-properties
:is-visible
:set-name^String name
:set-opacity^float opacity
:set-visible^boolean visible

(defn map-layer*
  ([]
    (MapLayer.))
  ([screen layer]
    (tiled-map-layer* screen layer)))
(defmacro map-layer
  [screen layer & options]
  `(let [^MapLayer object# (map-layer* ~screen ~layer)]
     (u/calls! object# ~@options)))
