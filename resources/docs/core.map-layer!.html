(map-layer! object k & options)

Calls a single method on a map-layer.

(map-layer! (map-layer screen "objects")
            :set-visible false)
Options
:get-name
:get-objects
:get-opacity
:get-properties
:is-visible
:set-name^String name
:set-opacity^float opacity
:set-visible^boolean visible
Source
(defmacro map-layer!
  [object k & options]
  `(u/call! ^MapLayer ~object ~k ~@options))
