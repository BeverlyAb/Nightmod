(remove-timer! screen id)

Stops and removes the timer associated with id, returning it or nil if not found.

Source
(defn remove-timer!
  [{:keys [update-fn!] :as screen} id]
  (when-let [timer (get-in screen [:timers id])]
    (.stop timer)
    (update-fn! update-in [:timers] dissoc id)
    timer))
