(circle-shape! object k & options)

Calls a single method on a circle-shape.

Options
:dispose
Needs to be called when the shape is no longer used, e.g. after a fixture was created based on the shape.
:get-child-count
Get the number of child primitives.
:get-position
:get-radius
Returns the radius of this shape
:get-type
Get the type of this shape. You can use this to down cast to the concrete shape.
:get-type
{@inheritDoc}
:set-position^Vector2 position
Sets the position of the shape
:set-radius^float radius
Sets the radius of this shape
Source
(defmacro circle-shape!
  [object k & options]
  `(u/call! ^CircleShape ~object ~k ~@options))
