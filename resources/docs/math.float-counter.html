(float-counter window-size & options)

Returns a FloatCounter.

Options
:put^float value
Add a value and update all fields.
:reset
Reset all values to their default value.
Source
(defn float-counter*
  [window-size]
  (FloatCounter. window-size))
(defmacro float-counter
  [window-size & options]
  `(u/calls! ^FloatCounter (float-counter* ~window-size) ~@options))
