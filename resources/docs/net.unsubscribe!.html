(unsubscribe! client & topics)

Unsubscribes client the given topics. The client is a hash map returned by the client function, or a play-clj screen map that contains a client hash map associated with the :network key.

(unsubscribe! screen :my-game-level-2)
Source
(defn unsubscribe!
  [client & topics]
  (doseq [t topics]
    (.unsubscribe (get-obj client :network :receiver) (str->bytes t))))
