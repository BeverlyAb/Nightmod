(table! entity k & options)

Calls a single method on a table.

Options
:add^Actor actor
Adds a new cell to the table with the specified actor.
:add^com.badlogic.gdx.scenes.scene2d.Actor[] actors
:add^String text
Adds a new cell with a label. This may only be called if {@link Table#Table(Skin)} or {@link #setSkin(Skin)} was used.
:add^String text, ^String label-style-name
Adds a new cell with a label. This may only be called if {@link Table#Table(Skin)} or {@link #setSkin(Skin)} was used.
:add^String text, ^String font-name, ^Color color
Adds a new cell with a label. This may only be called if {@link Table#Table(Skin)} or {@link #setSkin(Skin)} was used.
:add^String text, ^String font-name, ^String color-name
Adds a new cell with a label. This may only be called if {@link Table#Table(Skin)} or {@link #setSkin(Skin)} was used.
:add
Adds a cell without an actor.
:align^int align
Alignment of the logical table within the table actor. Set to {@link Align#center}, {@link Align#top}, {@link Align#bottom} , {@link Align#left}, {@link Align#right}, or any combination of those.
:background^Drawable background
:background^String drawable-name
:bottom
Adds {@link Align#bottom} and clears {@link Align#top} for the alignment of the logical table within the table actor.
:center
Sets the alignment of the logical table within the table actor to {@link Align#center}. This clears any other alignment.
:clear-children
Removes all actors and cells from the table.
:column-defaults^int column
Gets the cell values that will be used as the defaults for all cells in the specified column. Columns are indexed starting at 0.
:debug
:debug^Table.Debug debug
Turns debug lines on or off.
:debug-actor
Turns on actor debug lines.
:debug-all
:debug-cell
Turns on cell debug lines.
:debug-table
Turns on table debug lines.
:defaults
The cell values that will be used as the defaults for all cells.
:draw^Batch batch, ^float parent-alpha
:draw-debug^ShapeRenderer shapes
:get-align
:get-background
:get-cell^Actor actor
Returns the cell for the specified actor in this table, or null.
:get-cells
Returns the cells for this table.
:get-clip
:get-columns
:get-min-height
:get-min-width
:get-pad-bottom
:get-pad-bottom-value
:get-pad-left
:get-pad-left-value
:get-pad-right
:get-pad-right-value
:get-pad-top
:get-pad-top-value
:get-pad-x
Returns {@link #getPadLeft()} plus {@link #getPadRight()}.
:get-pad-y
Returns {@link #getPadTop()} plus {@link #getPadBottom()}.
:get-pref-height
:get-pref-width
:get-row^float y
Returns the row index for the y coordinate, or -1 if there are no cells.
:get-rows
:get-table-debug
:hit^float x, ^float y, ^boolean touchable
:invalidate
:layout
:left
Adds {@link Align#left} and clears {@link Align#right} for the alignment of the logical table within the table actor.
:pad^Value pad
Sets the padTop, padLeft, padBottom, and padRight around the table to the specified value.
:pad^Value top, ^Value left, ^Value bottom, ^Value right
:pad^float pad
Sets the padTop, padLeft, padBottom, and padRight around the table to the specified value.
:pad^float top, ^float left, ^float bottom, ^float right
:pad-bottom^Value pad-bottom
Padding at the bottom edge of the table.
:pad-bottom^float pad-bottom
Padding at the bottom edge of the table.
:pad-left^Value pad-left
Padding at the left edge of the table.
:pad-left^float pad-left
Padding at the left edge of the table.
:pad-right^Value pad-right
Padding at the right edge of the table.
:pad-right^float pad-right
Padding at the right edge of the table.
:pad-top^Value pad-top
Padding at the top edge of the table.
:pad-top^float pad-top
Padding at the top edge of the table.
:remove-actor^Actor actor
:reset
Removes all actors and cells from the table (same as {@link #clear()}) and additionally resets all table properties and cell, column, and row defaults.
:right
Adds {@link Align#right} and clears {@link Align#left} for the alignment of the logical table within the table actor.
:row
Indicates that subsequent cells should be added to a new row and returns the cell values that will be used as the defaults for all cells in the new row.
:set-background^String drawable-name
Sets the background drawable from the skin and adjusts the table's padding to match the background. This may only be called if {@link Table#Table(Skin)} or {@link #setSkin(Skin)} was used.
:set-background^Drawable background
:set-clip^boolean enabled
Causes the contents to be clipped if they exceed the table actor's bounds. Enabling clipping will set {@link #setTransform(boolean)} to true.
:set-debug^boolean enabled
:set-round^boolean round
If true (the default), positions and sizes are rounded to integers.
:set-skin^Skin skin
:stack^com.badlogic.gdx.scenes.scene2d.Actor[] actors
Adds a new cell to the table with the specified actors in a {@link Stack}.
:top
Adds {@link Align#top} and clears {@link Align#bottom} for the alignment of the logical table within the table actor.
