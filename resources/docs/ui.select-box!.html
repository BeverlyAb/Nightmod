(select-box! entity k & options)

Calls a single method on a select-box.

Options
:clear-items
:draw^Batch batch, ^float parent-alpha
:get-items
Returns the internal items array. If modified, {@link #setItems(Array)} must be called to reflect the changes.
:get-list
Returns the list shown when the select box is open.
:get-max-list-count
:get-pref-height
:get-pref-width
:get-scroll-pane
Returns the scroll pane containing the list that is shown when the select box is open.
:get-selected
Returns the first selected item, or null. For multiple selections use {@link SelectBox#getSelection()}.
:get-selected-index
:get-selection
Get the set of selected items, useful when multiple items are selected
:get-style
Returns the select box's style. Modifying the returned style may not have an effect until {@link #setStyle(SelectBoxStyle)} is called.
:hide-list
:layout
:set-disabled^boolean disabled
:set-items^java.lang.Object[] new-items
Set the backing Array that makes up the choices available in the SelectBox
:set-items^com.badlogic.gdx.utils.Array new-items
Sets the items visible in the select box.
:set-max-list-count^int max-list-count
Set the max number of items to display when the select box is opened. Set to 0 (the default) to display as many as fit in the stage height.
:set-selected^Object item
Sets the selection to only the passed item, if it is a possible choice, else selects the first item.
:set-selected-index^int index
Sets the selection to only the selected index.
:set-style^SelectBox.SelectBoxStyle style
:show-list
Source
(defmacro select-box!
  [entity k & options]
  `(u/call! ^SelectBox (u/get-obj ~entity :object) ~k ~@options))
