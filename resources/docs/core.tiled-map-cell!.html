(tiled-map-cell! object k & options)

Calls a single method on a tiled-map-cell.

(-> (tiled-map-layer screen "water")
    (tiled-map-cell 0 0)
    (tiled-map-cell! :set-rotation 90))
Options
:get-flip-horizontally
:get-flip-vertically
:get-rotation
:get-tile
:set-flip-horizontally^boolean flip-horizontally
Sets whether to flip the tile horizontally.
:set-flip-vertically^boolean flip-vertically
Sets whether to flip the tile vertically.
:set-rotation^int rotation
Sets the rotation of this cell, in degrees.
:set-tile^TiledMapTile tile
Sets the tile to be used for this cell.
Source
(defmacro tiled-map-cell!
  [object k & options]
  `(u/call! ^TiledMapTileLayer$Cell ~object ~k ~@options))
