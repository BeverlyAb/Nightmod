(chain-shape! object k & options)

Calls a single method on a chain-shape.

Options
:create-chain^float[] vertices
Create a chain with isolated end vertices.
:create-chain^com.badlogic.gdx.math.Vector2[] vertices
Create a chain with isolated end vertices.
:create-loop^float[] vertices
Create a loop. This automatically adjusts connectivity.
:create-loop^com.badlogic.gdx.math.Vector2[] vertices
Create a loop. This automatically adjusts connectivity.
:dispose
Needs to be called when the shape is no longer used, e.g. after a fixture was created based on the shape.
:get-child-count
Get the number of child primitives.
:get-radius
Returns the radius of this shape
:get-type
Get the type of this shape. You can use this to down cast to the concrete shape.
:get-type
:get-vertex^int index, ^Vector2 vertex
Returns the vertex at the given position.
:get-vertex-count
:is-looped
:set-next-vertex^Vector2 next-vertex
Establish connectivity to a vertex that follows the last vertex. Don't call this for loops.
:set-next-vertex^float next-vertex-x, ^float next-vertex-y
Establish connectivity to a vertex that follows the last vertex. Don't call this for loops.
:set-prev-vertex^Vector2 prev-vertex
Establish connectivity to a vertex that precedes the first vertex. Don't call this for loops.
:set-prev-vertex^float prev-vertex-x, ^float prev-vertex-y
Establish connectivity to a vertex that precedes the first vertex. Don't call this for loops.
:set-radius^float radius
Sets the radius of this shape

(defmacro chain-shape!
  [object k & options]
  `(u/call! ^ChainShape ~object ~k ~@options))
