(z! object z-val)

Sets only the z position of object. If object is a screen, the z position of the camera will be set.

Source
(defn z!
  [object z-val]
  (cond
    (isa? (type object) Vector3) (let [^Vector3 v object]
                                   (.set v (. v x) (. v y) z-val))
    :else (let [^Camera camera (u/get-obj object :camera)]
            (set! (. (. camera position) z) z-val)
            (.update camera))))
