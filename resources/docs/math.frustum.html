(frustum & options)

Returns a Frustum.

Options
:bounds-in-frustum^BoundingBox bounds
Returns whether the given {@link BoundingBox} is in the frustum.
:bounds-in-frustum^Vector3 center, ^Vector3 dimensions
Returns whether the given bounding box is in the frustum.
:bounds-in-frustum^float x, ^float y, ^float z, ^float half-width, ^float half-height, ^float half-depth
Returns whether the given bounding box is in the frustum.
:point-in-frustum^Vector3 point
Returns whether the point is in the frustum.
:point-in-frustum^float x, ^float y, ^float z
Returns whether the point is in the frustum.
:sphere-in-frustum^Vector3 center, ^float radius
Returns whether the given sphere is in the frustum.
:sphere-in-frustum^float x, ^float y, ^float z, ^float radius
Returns whether the given sphere is in the frustum.
:sphere-in-frustum-without-near-far^Vector3 center, ^float radius
Returns whether the given sphere is in the frustum not checking whether it is behind the near and far clipping plane.
:sphere-in-frustum-without-near-far^float x, ^float y, ^float z, ^float radius
Returns whether the given sphere is in the frustum not checking whether it is behind the near and far clipping plane.
:update^Matrix4 inverse-projection-view
Updates the clipping plane's based on the given inverse combined projection and view matrix, e.g. from an {@link OrthographicCamera} or {@link PerspectiveCamera}.
