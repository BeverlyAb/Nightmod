(matrix-4 values & options)

Returns a Matrix4.

Options
:cpy
:det
:det3x3
:extract4x3matrix^float[] dst
Copies the 4x3 upper-left sub-matrix into float array. The destination array is supposed to be a column major matrix.
:get-rotation^Quaternion rotation, ^boolean normalize-axes
Gets the rotation of this matrix.
:get-rotation^Quaternion rotation
Gets the rotation of this matrix.
:get-scale^Vector3 scale
:get-scale-x
:get-scale-xsquared
:get-scale-y
:get-scale-ysquared
:get-scale-z
:get-scale-zsquared
:get-translation^Vector3 position
:get-values
:idt
Sets the matrix to an identity matrix.
:inv
Inverts the matrix. Stores the result in this matrix.
:lerp^Matrix4 matrix, ^float alpha
Linearly interpolates between this matrix and the given matrix mixing by alpha
:mul^Matrix4 matrix
Postmultiplies this matrix with the given matrix, storing the result in this matrix. For example: 
 A.mul(B) results in A := AB.
:mul-left^Matrix4 matrix
Premultiplies this matrix with the given matrix, storing the result in this matrix. For example: 
 A.mulLeft(B) results in A := BA.
:rotate^Vector3 axis, ^float degrees
Postmultiplies this matrix with a (counter-clockwise) rotation matrix. Postmultiplication is also used by OpenGL ES' 1.x glTranslate/glRotate/glScale.

(defn matrix-4*
  ([]
    (Matrix4.))
  ([^floats values]
    (Matrix4. values)))
(defmacro matrix-4
  [values & options]
  `(u/calls! ^Matrix4 (matrix-4* ~values) ~@options))
