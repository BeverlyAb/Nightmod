(image-button! entity k & options)

Calls a single method on a image-button.

Options
:draw^Batch batch, ^float parent-alpha
:get-image
:get-image-cell
:get-style
:set-style^Button.ButtonStyle style
Source
(defmacro image-button!
  [entity k & options]
  `(u/call! ^ImageButton (u/get-obj ~entity :object) ~k ~@options))
