(edge-shape! object k & options)

Calls a single method on an edge-shape.

Options
:dispose
Needs to be called when the shape is no longer used, e.g. after a fixture was created based on the shape.
:get-child-count
Get the number of child primitives.
:get-radius
Returns the radius of this shape
:get-type
Get the type of this shape. You can use this to down cast to the concrete shape.
:get-type
:get-vertex0^Vector2 vec
:get-vertex1^Vector2 vec
:get-vertex2^Vector2 vec
:get-vertex3^Vector2 vec
:has-vertex0
:has-vertex3
:set^Vector2 v1, ^Vector2 v2
Set this as an isolated edge.
:set^float v1-x, ^float v1-y, ^float v2-x, ^float v2-y
Set this as an isolated edge.
:set-has-vertex0^boolean has-vertex0
:set-has-vertex3^boolean has-vertex3
:set-radius^float radius
Sets the radius of this shape
:set-vertex0^Vector2 vec
:set-vertex0^float x, ^float y
:set-vertex3^Vector2 vec
:set-vertex3^float x, ^float y
Source
(defmacro edge-shape!
  [object k & options]
  `(u/call! ^EdgeShape ~object ~k ~@options))
