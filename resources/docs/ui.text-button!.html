(text-button! entity k & options)

Calls a single method on a text-button.

Options
:draw^Batch batch, ^float parent-alpha
:get-label
:get-label-cell
:get-style
:get-text
:set-style^Button.ButtonStyle style
:set-text^String text
Source
(defmacro text-button!
  [entity k & options]
  `(u/call! ^TextButton (u/get-obj ~entity :object) ~k ~@options))
