(subscribe! screen & topics)

Subscribes the client the given topics. The screen is a hash map returned by the client function, or a play-clj screen map that contains a client hash map associated with the :network key.

(subscribe! screen :my-game-level-2)
Source
(defn subscribe!
  [screen & topics]
  (doseq [t topics]
    (.subscribe (u/get-obj screen :network :receiver) (u/str->bytes (name t)))))
