(fixture! object k & options)

Calls a single method on a fixture.

Options
:get-body
Get the parent body of this fixture. This is NULL if the fixture is not attached.
:get-density
Get the density of this fixture.
:get-filter-data
:get-friction
Get the coefficient of friction.
:get-restitution
Get the coefficient of restitution.
:get-shape
Returns the shape of this fixture
:get-type
Get the type of the child shape. You can use this to down cast to the concrete shape.
:get-user-data
:is-sensor
Is this fixture a sensor (non-solid)?
:refilter
Call this if you want to establish collision that was previously disabled by b2ContactFilter::ShouldCollide.
:set-density^float density
Set the density of this fixture. This will _not_ automatically adjust the mass of the body. You must call b2Body::ResetMassData to update the body's mass.
:set-filter-data^Filter filter
Set the contact filtering data. This will not update contacts until the next time step when either parent body is active and awake. This automatically calls Refilter.
:set-friction^float friction
Set the coefficient of friction.
:set-restitution^float restitution
Set the coefficient of restitution.
:set-sensor^boolean sensor
Set if this fixture is a sensor.
:set-user-data^Object user-data
Sets custom user data.
:test-point^Vector2 p
Test a point for containment in this fixture.
:test-point^float x, ^float y
Test a point for containment in this fixture.
Source
(defmacro fixture!
  [object k & options]
  `(u/call! ^Fixture ~object ~k ~@options))
