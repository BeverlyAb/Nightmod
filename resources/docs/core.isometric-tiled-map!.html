(isometric-tiled-map! screen k & options)

Calls a single method on an isometric-tiled-map.

Options
:dispose
:get-batch
:get-map
:get-unit-scale
:get-view-bounds
:render
:render^int[] layers
:render-object^MapObject object
:render-objects^MapLayer layer
:render-tile-layer^TiledMapTileLayer layer
:set-map^TiledMap map
:set-view^OrthographicCamera camera
:set-view^Matrix4 projection, ^float x, ^float y, ^float width, ^float height
Source
(defmacro isometric-tiled-map!
  [screen k & options]
  `(let [^IsometricTiledMapRenderer object# (u/get-obj ~screen :renderer)]
     (u/call! object# ~k ~@options)))
