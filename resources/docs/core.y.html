(y object)

Returns the y position of object. If object is a screen, the y position of the camera will be returned.

Source
(defn y
  [object]
  (cond
    (isa? (type object) Vector2) (. ^Vector2 object y)
    (isa? (type object) Vector3) (. ^Vector3 object y)
    :else (let [^Camera camera (u/get-obj object :camera)]
            (. (. camera position) y))))
