(label text arg & options)

Returns an entity based on Label.

(label "I'm a label" (color :white))
(label "I'm a label" (style :label (bitmap-font) (color :white)))
(label "I'm a label" (skin "uiskin.json"))

Provide any additional characters that should act as break characters when the label is wrapped. By default, only whitespace characters act as break chars.
:set-ellipsis^boolean ellipsis
When true the text will be truncated "..." if it does not fit within the width of the label. Default is false.

:set-wrap^boolean wrap
If false, the text will only wrap where it contains newlines (\n). The preferred size of the label will be the text bounds. If true, the text will word wrap using the width of the label. The preferred width of the label will be 0, it is expected that the something external will set the width of the label. Default is false.

When wrap is enabled, the label's preferred height depends on the width of the label. In some cases the parent of the label will need to layout twice: once to set the width of the label and a second time to adjust to the label's new preferred height.

(defn label*
  [^String text arg]
  (ActorEntity.
    (if (isa? (type arg) Color)
      (Label. text (style :label (g2d/bitmap-font) arg))
      (Label. text arg))))
(defmacro label
  [text arg & options]
  `(let [entity# (label* ~text ~arg)]
     (u/calls! ^Label (u/get-obj entity# :object) ~@options)
     entity#))
